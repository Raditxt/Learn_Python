# Learn Python

This is my personal Python learning repository. I document everything I learn here — not just the syntax, but the *why* behind each concept. The goal is to understand things from the ground up, which means no skipping fundamentals.

Writing things out forces me to actually understand them. If I can't explain it clearly in code and comments, I don't understand it yet.

## How I write these scripts

Every file follows a simple rule: explain it like you're teaching someone who has never seen code before. I use a lot of inline comments, simple variable names, and small examples before building up to the real thing.

This is basically the Feynman Technique applied to code. If a concept takes more than a few lines to explain, I usually don't understand it well enough yet.
